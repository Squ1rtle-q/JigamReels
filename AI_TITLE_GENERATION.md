# Автоматическое создание названий для видео через ИИ

## Описание функциональности

Реализована система автоматического создания интригующих и кликабельных названий для видео на основе содержания видео (транскрипции из Whisper).

## Компоненты

### 1. `utils/ai_helper.py` (новый файл)

Содержит две основные функции:

#### `generate_smart_title(transcription_text: str) -> str`
- Принимает текст транскрипции видео
- Отправляет запрос к ИИ модели через библиотеку `g4f`
- Использует модель `g4f.models.default` (автоматический выбор лучшей доступной модели)
- Возвращает сгенерированный заголовок (3-5 слов)
- При ошибке или таймауте возвращает fallback значение `"Video"`

**Промт для ИИ:**
```
Ты — эксперт по вирусному контенту в соцсетях. Проанализируй этот текст из видео: [текст].
На основе его смысла придумай один интригующий и кликабельный заголовок (3-5 слов).
Тон заголовка должен соответствовать видео (драматичный для кино, забавный для юмора, серьезный для советов).
Выдай ТОЛЬКО текст заголовка без кавычек и точек.
```

#### `sanitize_title_for_filename(title: str) -> str`
- Удаляет все символы, запрещенные в именах файлов Windows
- Заменяет пробелы на подчеркивания
- Удаляет лишние подчеркивания

### 2. `workers/worker.py` (обновлено)

Интегрирована логика автоматического создания названий:

1. **Сбор транскрипции:**
   - Для каждого обработанного видео/сегмента собирается текст из SRT файла
   - Весь текст из всех сегментов объединяется в одну строку

2. **Генерация названия:**
   - Вызывается `generate_smart_title(combined_transcription_text)`
   - Если название успешно сгенерировано - переходим к шагу 3
   - Если ошибка или таймаут - используется оригинальное имя файла

3. **Применение цензуры:**
   - Полученное название проходит через функцию `censor_words_in_text()`
   - Все запрещенные слова из списка `self.censor_words` цензурируются

4. **Очистка и переименование:**
   - Название очищается от недопустимых символов (Windows)
   - Используется функция `safe_filename()` для дополнительной валидации
   - Файл переименовывается с новым названием, сохраняя уникальный суффикс (timestamp)

5. **Новый метод `_extract_text_from_srt()`:**
   - Парсит SRT файлы и извлекает только текст субтитров
   - Убирает временные метки и индексы

## Процесс работы

```
1. Видео обрабатывается (нарезается, применяются эффекты)
   ↓
2. Генерируются субтитры через Whisper
   ↓
3. Текст из SRT файлов собирается в одну строку
   ↓
4. Запрос отправляется к ИИ (g4f)
   ↓
5. Применяется цензура к полученному названию
   ↓
6. Файл переименовывается с новым названием
   ↓
7. Финальное видео сохраняется с ИИ-названием
```

## Обработка ошибок

- **Таймаут ИИ**: используется fallback `"Video"`
- **Ошибка подключения**: используется fallback `"Video"`
- **Пустая транскрипция**: используется оригинальное имя файла
- **Некорректный результат ИИ**: используется fallback `"Video"`
- **Ошибка переименования**: логируется предупреждение, видео остается с исходным именем

## Требования

- Библиотека `g4f` версии 0.5.7.0 (уже установлена в `requirements.txt`)
- Включенный режим генерации субтитров (режим 'whisper' или 'srt_file')

## Примеры работы

### Входные данные
- Видео с речью: "Это невероятный способ готовить макароны, который изменит вашу жизнь..."

### Выходное названия (варианты)
- `Способ готовить макароны`
- `Революция в кулинарии`
- `Макароны по-новому`

### Файловая система
```
До:  video_processed_2025-02-26_18-45-30_abc12345.mp4
После: Способ_готовить_макароны_2025-02-26_18-45-30_abc12345.mp4
```

## Активация функции

Функция активируется автоматически при:
1. Включенном режиме генерации субтитров (subtitles mode ≠ 'none')
2. Успешной обработке видео
3. Наличии текста в субтитрах

## Логирование

Все события логируются в стандартный логгер Python:
- Успешная генерация названия
- Ошибки при запросе к ИИ
- Ошибки при переименовании файла
- Информация о цензуре

## Заметки о производительности

- Запрос к ИИ может занимать 5-15 секунд в зависимости от длины текста
- Рекомендуется вводить не более 2000 символов в ИИ для оптимизации
- Timeout установлен на 10 секунд (может быть настроен при необходимости)
